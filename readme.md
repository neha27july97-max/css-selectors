# CSS Selectors Assignment

## Project Overview

This project demonstrates the use of different CSS selectors to style HTML elements. The webpage contains headings, paragraphs, spans, and lists that are styled using element selectors, class selectors, ID selectors, and descendant selectors. The purpose of this project is to understand how CSS can be used to control the appearance of web page elements without using inline styles.

## Features:

* Orange background applied to the entire webpage.
* Blue-colored headings using a class selector.
* Lime background color applied using an ID selector.
* Red-colored text for specific words using the span element.
* Paragraph styling using element selectors.
* Special paragraph styling using an ID selector.
* Descendant selector used to style a specific word inside a paragraph.
* Ordered and unordered lists included.
* Styling applied to the word "here" in a list without using inline CSS, IDs,or classes.
